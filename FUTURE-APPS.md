# 📱 以后做更多应用的完整流程

## 首次设置（只做一次）

运行 `FIRST-TIME-SETUP.bat`，按照提示操作：
1. 输入GitHub用户名
2. 访问GitHub创建仓库
3. 推送代码

## 以后每次做新应用

只需两步：

### 步骤1：修改代码
修改 `yuv_converter_android.py` 或创建新的Python文件

### 步骤2：重新编译
运行 `UPDATE.bat`，等待5-10分钟，下载新APK

## 🎯 好处

- ✅ 只需要首次设置一次
- ✅ 以后只需双击一个文件
- ✅ 自动编译，无需等待
- ✅ 完全免费
- ✅ GitHub Actions会帮你搞定一切

## 📦 生成的文件位置

- APK在：仓库的Actions页面 → Artifacts
- 或者在Releases页面直接下载
