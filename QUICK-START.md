# 🚀 超简单获取APK指南

## 一键上传脚本（我帮你写好了）

我已经为你准备好了所有文件，现在只需要这3步：

### 第1步：创建GitHub仓库（30秒）
1. 访问：https://github.com/new
2. 仓库名填：`yuv-converter-android`
3. 选 Public（公开）
4. 点击 "Create repository"

### 第2步：上传文件（1分钟）
1. 点击 "uploading an existing file"
2. 打开文件夹：`C:\Users\yang_\Desktop\RGB2YUV-Android`
3. 全选所有文件，拖拽到网页
4. 点击 "Commit changes"

### 第3步：下载APK（等待5-10分钟）
1. 等待5-10分钟
2. 访问仓库的 "Actions" 标签
3. 查看编译状态
4. 编译完成后，点击 "Artifacts" 下载APK

## ⏱️ 总时间：约10-12分钟

这是我能做的最简单方案了。所有准备工作我都完成了！
